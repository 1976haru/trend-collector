# public/assets/

법무부(MOJ) 스타일 헤더에서 사용하는 정적 자산을 두는 폴더입니다.

## 로고 파일

`MOJHeader` 는 다음 경로에서 로고를 자동으로 찾습니다.

```
public/assets/moj-logo.png
```

- **권장 크기**: 가로 96px × 세로 96px 내외 (정사각 또는 가로형 가능)
- **포맷**: PNG (투명 배경)
- **권장 비율**: 1:1 또는 4:1 이내

이 파일이 존재하지 않으면, 헤더는 자동으로 **태극 풍 원형 + 한글 "법" 텍스트** fallback 을 표시합니다.
이 fallback 만으로도 시스템은 정상 동작합니다.

## 외부 핫링크 금지

법무부 공식 홈페이지(http://www.moj.go.kr)의 이미지를 직접 hot-link 로 참조하지 마세요.
대상 사이트의 사칭 위험과 법적 리스크가 있습니다.
필요한 경우 부서 내부에서 적법하게 확보한 이미지를 위 경로에 배치하세요.

## 추가 자산 슬롯

향후 추가 가능:
- `moj-symbol.png` — 마크 단독 (favicon 용)
- `moj-banner.png` — 헤더 가로 배너
