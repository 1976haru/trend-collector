# 🗺 ROADMAP — 공무원 / 업무용 기능 방향

Trend Collector 를 단순 뉴스 수집기에서 **트렌드 수집 + 요약 + PDF/메일 발송 기반 자동 리포트 시스템**, 더 나아가 **일일 업무보고용 도구**로 확장하기 위한 계획입니다.

---

## 1단계 — 보고서 기반 (이번 라운드 일부 반영)

| 항목 | 상태 | 메모 |
|------|------|------|
| "요약 / 주요 이슈 / 시사점 / 참고 링크" 보고서 템플릿 | ✅ 1차 반영 | `src/utils/pdfUtils.js` |
| 중앙언론 / 지방언론 / 인터넷언론 구분 | ✅ 1차 반영 | `src/constants/mediaList.js` `MEDIA_TIER` |
| 지역별 뉴스 분류 (서울·강원·충청·호남·영남·제주) | ✅ 1차 반영 | `mediaList.js` `MEDIA_BY_CATEGORY` + `classifyRegion()` |
| 키워드 급상승 감지 | ✅ 1차 반영 | `src/utils/filterUtils.js` `detectTrendingKeywords()` |
| 중복 기사 제거 | ✅ 1차 반영 | URL + 정규화된 제목 기반 |
| 광고/홍보성 기사 필터링 | ✅ 1차 반영 | `DEFAULT_AD_KEYWORDS`, on/off 토글 |
| 일일 업무보고용 PDF 자동 생성 | 🟡 일부 | 수동 버튼은 동작. 스케줄 트리거에서 자동 생성은 다음 단계 |

---

## 2단계 — 정책 / 행정 특화

- **정책 / 행정 / 지자체 키워드 프리셋**
  - "정부, 국정, 국무회의, 대통령, 국회, 예산, 법안, 정책, 행정, 지자체, 복지, 교육, 보건, 환경, 교통, 경제, 부동산, 청년, 일자리, 재난" 등은 이미 `PRESET_KEYWORDS` 에 반영.
  - 부처별(보건복지부 / 행정안전부 / 국토교통부 …) 프리셋 그룹 도입.
- **지자체별 빠른 필터** — 광역시·도 17개를 토글 칩으로 제공.
- **부고 / 인사 / 동정 자동 분류** (공무원 보고에서 자주 빠지는 카테고리).
- **법률·고시·공고 RSS 추가** — `법제처 국가법령정보`, `정부24` 등.

---

## 3단계 — 발송 / 워크플로우

- **발송 시간 + 채널 매트릭스**
  - 09:00 — 메일 (팀장단)
  - 12:00 — 카카오톡 (실무자)
  - 18:00 — PDF 만 저장(메일 미발송)
- **사용자별 키워드 / 수신자 그룹** — 부서별 별도 설정 저장.
- **결재 라인 머리말 / 보고서 표지** 입력란.
- **EmailJS 첨부 PDF 발송** — 현재는 본문 텍스트.
- **카카오 알림톡** (비즈니스 채널 연동, 유료).

---

## 4단계 — 백그라운드 자동화

브라우저 의존도를 없애기 위한 단계.

- **GitHub Actions cron** — 매일 09:00 KST 빌드 → 정적 산출물 PDF 를 Pages 또는 Releases 에 업로드.
- **Cloudflare Workers / AWS Lambda** — RSS 수집 + Anthropic API 요약 + 메일 발송을 서버리스로.
- **Supabase / Cloudflare D1** — 사용자 / 키워드 / 발송 이력 저장.
  - 스키마 초안: `users(id, email)`, `keyword_groups(id, user_id, name)`, `keywords(id, group_id, term, kind)`, `reports(id, group_id, sent_at, channel, count)`.

---

## 5단계 — 분석 고도화

- **Claude / Anthropic API 감성 분석** (긍정 / 부정 / 중립 + 톤 강도).
- **이슈 클러스터링** — 동일 사건을 다룬 기사 묶기 (제목 임베딩 → 코사인 유사도).
- **시기별 / 매체별 보도 추이 그래프**.
- **원문 본문 자동 요약** — 1줄 / 3줄 / 7줄 길이 선택.
