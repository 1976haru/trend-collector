# 🗺 ROADMAP — 공무원 / 업무용 확장

내부 직원 MVP 이후의 확장 방향.

---

## 1단계 — MVP (완료)

- 단일 비밀번호 로그인, 외부 공개 X.
- 키워드 / 제외 / 수신자 서버 공유 저장.
- 즉시 수집 + 다중 스케줄 모드 (daily / interval / off).
- 언론 유형 분류, 감정 분석(긍·부·중립), 급상승 감지, 중복 묶기, 자동 요약 문장.
- PDF 인쇄, SMTP 메일 발송, 알림 트리거 플래그.
- 모바일 반응형 + PWA.
- Render Web Service 단일 서비스 배포.

---

## 2단계 — 분석·발송 고도화 (P1·P2)

- **PDF 파일 첨부 메일** — Puppeteer + @sparticuz/chromium 로 서버에서 PDF 바이너리 생성.
- **긴급 알림 메일 분리** — 부정 50% 이상 / 급상승 / 중앙언론 / 정부·공공기관 트리거 시 별도 짧은 알림.
- **일·주·월 분석 탭** — 기간별 키워드/긍부정/매체 변화 시계열.
- **수신자 그룹** — 팀장단 / 실무자 / 전체.
- **검색·필터** — 리포트 내 키워드/언론사 검색.

---

## 3단계 — 정책·행정 특화

- **부처별 / 지자체별 키워드 프리셋** — 보건복지부 / 행정안전부 / 17개 광역지자체.
- **법령·고시·공고 RSS 추가** — 법제처 국가법령정보, 정부24 공고.
- **부고 / 인사 / 동정 자동 분류**.
- 매체 화이트/블랙리스트 (특정 언론사만 / 제외).

---

## 4단계 — 다중 부서 / 사용자별 계정

- ADMIN_PASSWORD 1개 → **부서별 토큰** (회원가입 X, 운영자 발급).
- 부서별 키워드/수신자 묶음.
- **기관별 권한 관리** — 읽기 전용 / 발송 권한.
- 발송 이력 / 열람 로그 (감사 추적).

---

## 5단계 — 데이터·인프라 정식화

- **DB 마이그레이션** — JSON 파일 → SQLite 또는 Postgres.
  - SQLite: 작은 운영 환경에 적합, Render Disk 1개로 OK.
  - Postgres: Render Postgres / Supabase 무료 티어부터 가능. 다중 인스턴스 / 백업 / 점진적 분석 쿼리에 유리.
- **장기 데이터** 6개월~1년 추세 분석.
- 컨테이너화 (Dockerfile) + 멀티 클라우드 (Cloud Run / Fly.io) 옵션.

---

## 6단계 — AI / 외부 연동

- **LLM 기반 고급 요약** — Claude Sonnet 또는 OpenAI 로 1~3줄 요약 + “오늘의 핵심 이슈 5개” 자동 발문.
- **이슈 임베딩 클러스터링** — 제목/본문 임베딩 → 코사인 유사도 기반 동일 사건 묶기.
- **실제 뉴스 API 연동** — Google News RSS 외에 NAVER 뉴스 검색 API, 빅카인즈 등 추가 소스.
- **카카오톡 비즈 알림톡** — 유료 채널 연동.
